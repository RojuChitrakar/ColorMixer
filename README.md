

# 🎨 Color Mixer

A simple and fun **React + Vite** project that lets you mix RGB colors interactively and save your favorite color combinations using Local Storage.

## 🚀 Features

- 🎚️ Adjust **Red**, **Green**, and **Blue** sliders to create your custom color  
- 🎨 Live color preview updates instantly  
- 💾 Save and load your last color automatically using **Local Storage**  
- 💻 Clean and responsive design  

---

## 🧠 Technologies Used

- ⚛️ **React (Vite)**
- 🟦 **JavaScript (ES6)**
- 🎨 **CSS3 (External Stylesheet)**
- 💾 **Local Storage API**

## ⚙️ How to Run
# Clone this repository
```bash
git clone https://github.com/RojuChitrakar/color-mixer.git
```
# Navigate into the project folder
```bash
cd color-mixer
```
# Install dependencies
```bash
npm install
```
# Start the development server
```bash
npm run dev
```
# Open your browser
```bash
Go to 👉 http://localhost:5173/
```
---
## 🧠 How It Works
   Adjust the Red, Green, and Blue sliders to create a custom color.
   
   The color preview updates live as you move the sliders.
   
   Click Save Color 💾 to store the current color in your browser’s localStorage.
   
   When you refresh or reopen the app, your last saved color appears automatically.
